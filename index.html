Projects

National Car Dealership Review Portal

I built a fully-featured web portal for a national car dealership network as part of my IBM capstone project. The portal enables customers to easily search for dealerships by location, read reviews, and allow logged-in users to submit new reviews. Additionally, it provides insights into the sentiment of these reviews, making it easier for users to gauge customer experiences.

Weather Forecast Webpage (7Timer API)

In this project, I developed an European weather forecast webpage. By integrating the 7Timer API, I created a user-friendly interface that allows users to check 7-day weather forecasts for majo cities. This feature helps engage users and encourages longer visits. I showcased the project to demonstrate my skills in API integration and web development.

Books Expense-Tracking Web App

I developed a sample expense-tracking web app for a fictional book distribution company, Rumi Press, as part of my learning journey. Using Python and Django, I built data models, set up Django Admin, and created visual reports. This app demonstrates how teams can effectively manage book distribution expenses. I published the project on GitHub
